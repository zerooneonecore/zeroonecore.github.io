The boot sequence

Makefiles and config files were scattered everywhere, and this wasn't seeming to yield any results. What I did know is that the kernel has to interface in some way with the BIOS/firmware, and decided to try searching "ROM, BIOS, firmware". An install script within the architecture specific directory for ia64 made a reference to "vmlinuz" and "System.map"
